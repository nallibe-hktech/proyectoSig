4. DATOS QUE APORTA CADA API A LA HERRAMIENTA
Cada API integrada proporciona datos específicos que alimentan diferentes módulos de la plataforma:
🔷 BIZNEO (Sistema de Gestión de RRHH)
Datos que aporta: • Lista de usuarios / empleados (ID, Email, Nombre, Foto) • Ausencias y vacaciones (fechas de inicio/fin, tipo de ausencia) • Información de personal activo  Qué hace la herramienta con estos datos: ✓ Mantiene un registro actualizado de todos los empleados ✓ Sincroniza automáticamente cambios en empleados (altas, bajas, cambios) ✓ Calcula disponibilidad real de personal (excluye vacaciones/bajas) ✓ Asigna tareas/visitas solo a empleados disponibles ✓ Integra datos en reportes de productividad por persona
🔷 CELERO ONE (Sistema de Gestión de Proyectos y Visitas)
Datos que aporta: • Proyectos activos (ID, nombre, cliente, presupuesto) • Visitas planificadas y realizadas • Feedback de clientes • Información geográfica (ubicaciones) • Asignación de empleados a proyectos  Qué hace la herramienta con estos datos: ✓ Carga en tiempo real información de proyectos ✓ Sincroniza estado de visitas (planificada, realizada, cancelada) ✓ Valida datos antes de calcular pagos y facturación ✓ Usa geolocalización para validar incidencias ✓ Base para calcular métricas de efectividad
🔷 INTRATIME (Sistema de Registro de Tiempos - PENDIENTE)
Datos que aporta: • Horas trabajadas por empleado • Asistencia y puntualidad • Tiempo dedicado a proyectos  Qué hace la herramienta con estos datos: ✓ Valida horas registradas vs. horas facturables ✓ Detecta discrepancias (horas registradas vs. visitas realizadas) ✓ Calcula bases de pago por proyecto
🔷 PAYHAWK (Sistema de Gastos Corporativos - PENDIENTE)
Datos que aporta: • Gastos realizados por empleados (restaurante, hotel, transporte) • Recibos y justificantes • Datos del empleado y proyecto asociado  Qué hace la herramienta con estos datos: ✓ Registra gastos por empleado y proyecto ✓ Valida que los gastos cumplan políticas de empresa ✓ Incluye gastos en cálculo de dietas/reembolsos ✓ Genera informes de gastos por proyecto para facturación
🔷 A3 NÓMINA (Sistema de Nóminas - BLOQUEADO)
Datos que aporta: • Estructura de nómina por empleado • Conceptos salariales (base, complementos, retenciones) • Datos de plantilla  Qué hace la herramienta con estos datos: ✓ Valida si pagos a empleados cumplen estructura de nómina ✓ Integra conceptos extras (dietas, incentivos) en nómina ✓ Genera base para previsión de costos salariales
🔷 A3 ERP (Sistema de Facturación - BLOQUEADO)
Datos que aporta: • Clientes de la empresa • Tarifas y políticas de facturación • Cuentas contables  Qué hace la herramienta con estos datos: ✓ Valida clientes antes de facturar ✓ Aplica tarifas correctas según cliente/proyecto ✓ Genera facturas con datos válidos ✓ Integración con contabilidad
🔷 TRAVELPERK (Sistema de Viajes - PENDIENTE)
Datos que aporta: • Viajes reservados y realizados • Costos de transporte (vuelos, hoteles) • Empleado, proyecto y fechas  Qué hace la herramienta con estos datos: ✓ Registra costos de viaje en proyecto ✓ Valida que viajes sean justificables ✓ Suma costos de viaje en presupuesto del proyecto
🔷 GALÁN (Sistema de Logística y Almacenes - BLOQUEADO)
Datos que aporta: • Stock de productos en almacén • Entradas y salidas diarias • Costos logísticos (manipulación, almacenaje, transporte) • Información de transporte y entregas  Qué hace la herramienta con estos datos: ✓ Sincroniza inventario en tiempo real ✓ Valida disponibilidad de stock antes de compromisos ✓ Incluye costos logísticos en facturas a clientes ✓ Genera alertas de stock bajo
🔷 SGPV (Sistema de Gestión de Planificación - PENDIENTE)
Datos que aporta: • Datos de planificación y control de proyectos • Hitos y deadlines  Qué hace la herramienta con estos datos: ✓ Valida cumplimiento de plazos ✓ Genera alertas de retraso ✓ Integra en reportes ejecutivos
🔷 MEDIAPOST (Sistema de Distribución - BLOQUEADO)
Datos que aporta: • Información de envíos y entregas  Qué hace la herramienta con estos datos: ✓ Registra seguimiento de entregas ✓ Valida cumplimiento de entregas ✓ Incluye costos de distribución
5. FUNCIONALIDADES PRINCIPALES DE LA HERRAMIENTA
La herramienta SIG-ES integrará todos estos datos para automatizar procesos críticos:
A) GESTIÓN OPERATIVA (Operaciones + Proyectos)
• Consolida información de proyectos (Celero) y empleados (Bizneo) • Visualiza qué empleados están disponibles y para qué proyectos • Asigna tareas considerando vacaciones, ausencias y tiempos registrados • Detecta problemas: visitas planificadas vs. horas trabajadas • Genera alertas si hay desviaciones
B) CÁLCULO DE PAGOS Y DIETAS (RRHH + Finanzas)
• Combina datos de:   - Bizneo: empleados y ausencias   - Intratime: horas trabajadas   - Celero: proyectos donde trabajó   - PayHawk: gastos realizados   - A3 Nómina: estructura salarial • Automatiza cálculo de:   - Dietas (según horas trabajadas)   - Kilometraje   - Reembolsos de gastos   - Gratificaciones extras   - Incentivos por visitas realizadas • Valida que todo cumpla políticas y nómina
C) FACTURACIÓN A CLIENTES (Finanzas + Operaciones)
• Integra datos de:   - Celero: visitas realizadas, proyectos   - A3 ERP: tarifas y clientes   - Galán: costos logísticos   - TravelPerk: costos de viaje • Calcula automáticamente:   - Precio por visita/proyecto   - Gastos asociados al cliente   - Costos logísticos imputables   - Márgenes por cliente • Genera facturas con A3 ERP
D) REPORTING Y ANÁLISIS (BI/Dirección)
• Consolida TODOS los datos en dashboards • KPIs por proyecto:   - Rentabilidad (ingresos vs. costos)   - Desviaciones presupuestarias   - Productividad por empleado   - Tasa de cumplimiento de proyectos • KPIs por cliente:   - Márgenes   - Cumplimiento de SLA • Power BI: exporta datos limpios para análisis avanzado
